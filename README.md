# Chronoide(크로노이드)

Chrono(시간, 일정) + Guide(안내)의 합성어입니다.

해당 프로젝트는 일정 관리를 위해 만들어졌습니다.

사용자는 일정을 등록하고 개략적인 작업 일수를 산정합니다.

버퍼 일정을 추가하거나 시작과 마감 일자를 선택해서 작업 일정을 쉽게 생성할 수 있습니다.
